Описание по выполнению требований:

1. Код был написан в соответсвии с текущими знаниями С++.
2. Был реализован многофайловый проект без нарушения композиции и инкапсуляции.
3. Был испоьзован контейнер array.
4. Структура кода соответсвует выполнению и последовательности вызовов.
5. Были подключены библиотеки SDL2, SDL2_Image и SDL_ttf.
6. Программа выполненна для игрового взамодействия - ввод игрока, взаимодействие внутриигровых элементов (шар и искуственный интелект), проверка столкновений, индикация счета игрока и ИИ, отрисовка игровых обьектов.
7. Был использован паттерн "фасад" - основной класс Game делегирует работу в соотвествующие классы и следит за жизненным циклом объектов. Компоненты не знают о существовании класса в котором используются не нарушая композицию.
8. Проект собран для выполнения под Win32.
